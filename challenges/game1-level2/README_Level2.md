
# 🧠 Thử thách Level 2 – Viết lại xử lý kết quả Kéo Búa Bao

## Mục tiêu:
Bạn sẽ viết lại phần xử lý kết quả trong file `script.js`, bao gồm:
- Cập nhật điểm số người chơi và máy
- Hiển thị kết quả: "Bạn thắng!", "Bạn thua!", "Hòa rồi!"
- Cập nhật kỷ lục nếu đạt điểm cao.

## Việc cần làm:
1. Mở `script.js`
2. Tìm các đoạn `// TODO`
3. Khi người chơi đạt điểm cao hơn


## ✅ Kết quả mong muốn:
- Game hoạt động bình thường
- Hiển thị đúng kết quả
- Cập nhật kỷ lục nếu người chơi đạt điểm cao
